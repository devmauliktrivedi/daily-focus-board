# Daily Focus Board

A tiny beginner-friendly project for learning Git and GitHub.

## What it does

- saves one main focus
- lets you add tasks
- marks tasks as done
- stores everything in your browser with local storage

## How to open it

Open `index.html` in your browser.

## Files

- `index.html` - page structure
- `style.css` - page design
- `script.js` - app logic

## Good practice changes you can try later

- change colors
- change the title
- add a dark mode
- add a task counter
- add a date line
